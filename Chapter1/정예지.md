# Chapter1: 컴퓨터 구조 시작하기

## 1. 컴퓨터가 이해하는 정보

컴퓨터는 명령어를 처리하는 기계

- **데이터**: 컴퓨터가 이해하는 정적인 정보 (ex, 숫자, 문자, 동영상, 사진 ... )
- **명령어**: 데이터를 움직이고, 컴퓨터를 작동시키는 정보

## 2. 컴퓨터의 4가지 핵심 부품

다음 부품들은 모두 **메인 보드 (마더보드)** 라는 판에 연결된다.

> 메인보드는 여러 컴퓨터 부품을 부착할 수 있는 슬롯과 연결 단자가 존재한다.
</br>

메인보드에 연결된 부품들은 **버스**라는 통로를 통해 서로 정보를 주고 받을 수 있다.
컴퓨터의 네가지 부품을 연결하는 가장 중요한 버스는 **시스템 버스**이다.

> 시스템 버스는 **주소 버스, 데이터 버스, 제어버스**로 구성되어 있다.
>
> - **주소 버스**: 주로를 주고 받는 통로
> - **데이터 버스**: 명령어, 데이터를 주고 받는 통로
> - **제어 버스**: 제어 신호를 주고 받는 통로, CPU는 제어버스를 통해 제어 신호를 내보낸다.


### CPU

메모리에 저장된 데이터를 읽고, 해석하고, 실행하는 부품

- **ALU**: 컴퓨터 내부에서 수행되는 대부분의 계산을 수행
- **레지스터**: CPU 내부에서 임시로 값을 저장
- **제어장치**: 제어 신호를 내보내고 명령어를 해석하는 장치

> 제어신호:
> 컴퓨터 부품들을 관리하고 작동시키기 위한 전기신호
>
> - 메모리 읽기 제어신호: CPU가 메모리에 저장된 값을 읽고 싶은 경우
> - 메모리 쓰기 제어신호: CPU가 메모리에 어떤 값을 저장하고 싶은 경우

### 주기억장치(메모리)

**현재 실행되는** 프로그램의 데이터, 명령어를 저장하는 부품
즉, 현재 실행되는 프로그램은 반드시 메모리에 저장되어 있다.

- **단점**: 가격이 비싸 저장 용량이 적고, 전원이 꺼지면 저장된 내용을 잃는다

> 주소: 메모리가 저장된 값에 빠르고 효율적으로 접근하기 위해 사용되는 개념

### 보조기억장치

- 주기억장치보다 크기가 크고, 전원이 꺼져도 저장된 내용을 잃지 않는 저장 장치
- 주로 현재 실행되는 프로그램은 주기억장치, 보관할 프로그램을 저장은 보조기억 장치에서 맡는다

### 입출력 장치

컴퓨터 외부에 연결되어 컴퓨터 내부와 정보를 교환하는 장치
