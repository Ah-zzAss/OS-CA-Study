# 컴퓨터 구조의 큰 그림

## 컴퓨터가 이해하는 정보

1. 데이터 : 숫자, 문자, 이미지, 동영상
2. 명령어 : instruction, ex) add $s1, $s2, $s3

## 컴퓨터의 4가지 핵심 부품

1. CPU
    - 메모리에 저장된 명령어를 읽고 해석하고 실행
    - ALU(산술논리연산장치) : 연산하는 장치
    - 레지스터 : CPU 내부의 임시 저장 장치
    - 제어장치 : 제어신호를 내보내고 명령어를 해석하는 장치
2. 메모리
    - 현재 실행되는 프로그램의 명령어와 데이터를 저장
    - 메모리 내에 주소가 존재하여 원하는 위치에 접근 가능
    - 휘발성
3. 보조기억장치
    - 보관할 데이터를 저장
    - 비휘발성
4. 입출력장치
    - 컴퓨터 내부와 정보를 교환하는 장치
    - 마이크, 스피커, 프린터, 마우스, 키보드 등
