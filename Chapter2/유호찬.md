# Chapter 2. 데이터

# 0과 1로 숫자를 표현하는 방법

## 정보 단위

- 비트 : 0과 1을 나타내는 가장 작은 정보 단위
- n개의 비트로 $2^n$가지 정보를 표현 가능

## 이진법

- 컴퓨터는 0과 1만으로 이진법을 사용
- 이진법을 표기하기 위해 아래첨자($1000_{(2)}$)를 붙이거나 0b를 붙인다.

### 이진법의 음수 표현

- 2의 보수
    1. 사전적 의미 : 어떤 수를 그보다 큰 $2^n$에서 뺀 값
    2. bit flip하고 그 값에 1을 더한 값
- 플래그(flag)를 사용하여 음수와 양수를 구분

### 십육진법

- 십진수의 10, 11, 12, 13, 14, 15를 A, B, C, D, E, F로 표기
- 십육진법을 표기하기 위해 아래첨자($F_{(16)}$)를 붙이거나 0x를 붙인다.
- 십육진수 ↔ 이진수
    - 십육진수를 이루는 숫자 하나를 표현하려면 4비트가 필요. $2^4 = 16$
    - 십육진수의 각 숫자를 이진수 4비트로 표현하여 변환

# 0과 1로 문자를 표현하는 방법

## 문자 집합과 인코딩

1. 문자 집합 : 컴퓨터가 인식하고 표현할 수 있는 문자의 모음
2. 문자 인코딩 : 문자를 0과 1로 변환하는 과정
3. 문자 디코딩 : 문자 코드를 문자로 변환하는 과정

2바이트 크기로 한글을 인코딩 할 수 있는 완성형 인코딩 방식

### 유니코드와 UTF-8

- 대부분의 나라의 문자, 특수문자 등을 표현할 수 있는 문자 집합
- 현대 문자를 표현할 때가장많이 사용되는 표준 문자 집합
- UTF-8, UTF-16, UTF-32는 유니코드 문자에 부여된 값을 인코딩하는 방식
